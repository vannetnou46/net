Data Types.

Data Types ឬ ប្រភេទទិន្នន័យ គឺជាការកំណត់ថាតើអថេរ (Variable) មួយអាចផ្ទុកទិន្នន័យប្រភេទណាខ្លះ។ ក្នុងភាសា PHP យើងមិនចាំបាច់ប្រកាសប្រភេទអថេរទេ ព្រោះ PHP គឺជា Dynamic Typed Language ដែលមាន ន័យថាប្រភេទទិន្នន័យនឹងត្រូវបានកំណត់ដោយស្វ័យប្រវត្តិ តាមតម្លៃដែលផ្តល់ឲ្យ។

  • PHP គាំទ្រប្រភេទទិន្នន័យចំនួន ៨ ប្រភេទ,
  • ប្រភេទទិន្នន័យត្រូវបានកំណត់ដោយស្វ័យប្រវត្តិ (Dynamic Typing),
  • អាចប្រើ function var_dump() ដើម្បីពិនិត្យប្រភេទ និងតម្លៃរបស់អថេរ,
  • <.?php
    // Integer Types
    • $age = 25; // Decimal (ទសភាគ)
    $binary = 0b1010; // Binary (គោលពីរ) = 10
    $octal = 0123; // Octal (គោលប្រាំបី) = 83
    $hexadecimal = 0x1A; // Hexadecimal (គោលដប់ប្រាំមួយ) = 26
    echo "Age: $age";
    echo "Binary 0b1010 = $binary";
    echo "Octal 0123 = $octal";
    echo "Hexadecimal 0x1A = $hexadecimal";
    echo "Max Integer: " . PHP_INT_MAX . "";
    echo "Min Integer: " . PHP_INT_MIN;
  • ?>
  • លទ្ទិផល.
    • Age: 25
    Binary 0b1010 = 10
    Octal 0123 = 83
    Hexadecimal 0x1A = 26
    Max Integer: 9223372036854775807
    Min Integer: -9223372036854775808

+ Scalar Types.
(ប្រភេទទិន្នន័យមួលដ្ឋាន)

Scalar Types គឺជាប្រភេទទិន្នន័យដែលផ្ទុកតម្លៃតែមួយ (single value)។ មាន ៤ ប្រភេទគឺ៖

1. Integer (ចំនួនគត់),

ប្រើសម្រាប់ផ្ទុកចំនួនគត់ (គ្មានខ្ទង់ទសភាគ)។

2. សេចក្តីផ្តើមអំពី
Data Types.

Data Types ឬ ប្រភេទទិន្នន័យ គឺជាការកំណត់ថាតើអថេរ (Variable) មួយអាចផ្ទុកទិន្នន័យប្រភេទណាខ្លះ។ ក្នុងភាសា PHP យើងមិនចាំបាច់ប្រកាសប្រភេទអថេរទេ ព្រោះ PHP គឺជា Dynamic Typed Language ដែលមាន ន័យថាប្រភេទទិន្នន័យនឹងត្រូវបានកំណត់ដោយស្វ័យប្រវត្តិ តាមតម្លៃដែលផ្តល់ឲ្យ។

  • PHP គាំទ្រប្រភេទទិន្នន័យចំនួន ៨ ប្រភេទ,
  • ប្រភេទទិន្នន័យត្រូវបានកំណត់ដោយស្វ័យប្រវត្តិ (Dynamic Typing),
  • អាចប្រើ function var_dump() ដើម្បីពិនិត្យប្រភេទ និងតម្លៃរបស់អថេរ,
  • <.?php
    // Boolean Types
    • $isActive = true;
    $isLoggedIn = false;
    $hasPermission = 1; // 1 ត្រូវបានចាត់ទុកជា true
    $isEmpty = 0; // 0 ត្រូវបានចាត់ទុកជា false
    echo "Is Active: " . ($isActive ? 'Yes' : 'No') . "br";
    echo "Is Logged In: " . ($isLoggedIn ? 'Yes' : 'No') . "br";
    echo "Has Permission: " . ($hasPermission ? 'Yes' : 'No') . "br";
    echo "Is Empty: " . ($isEmpty ? 'Yes' : 'No');?>
  • ?>
  • លទ្ទិផល.
    • Is Active: Yes
    Is Logged In: No
    Has Permission: Yes
    Is Empty: No

3. Boolean.
(តម្លៃពិត/មិនពិត)

ប្រើសម្រាប់ផ្ទុកតម្លៃឡូជីខល៖ true ឬ false។

4. String.
(អក្សរ/អត្ថបទ)

ប្រើសម្រាប់ផ្ទុកអត្ថបទ ឬតួអក្សរ។


+Noted គន្លឹះ៖ ការប្រើ double quotes អនុញ្ញាតឲ្យយើងបញ្ចូល variable ដោយផ្ទាល់ក្នុង string ខណៈ single quotes មិនអាច។

  • <.?php
    // String Types
    • $singleQuote = 'Hello PHP'; // Single quotes
    $doubleQuote = "Hello PHP"; // Double quotes (អាចប្រើ variable)
    $name = "Samrech";
    $greeting = "Welcome, $name!"; // Variable parsing
    $heredoc = EOT
    This is a heredoc string.
    It can contain multiple lines.
    Variables like $name are parsed.
    EOT;
    echo "Single Quote: $singleQuote.br";
    echo "Double Quote: $doubleQuote.br";
    echo "Greeting: $greeting.br";
    echo "Heredoc: $heredoc";
  • ?>
  • លទ្ទិផល.
    • Single Quote: Hello PHP
    Double Quote: Hello PHP
    Greeting: Welcome, Samrech!
    Heredoc: This is a heredoc string. It can contain multiple lines. Variables like Samrech are parsed.
  • <.?php
    // Integer Types
    • $age = 25; // Decimal (ទសភាគ)
    $binary = 0b1010; // Binary (គោលពីរ) = 10
    $octal = 0123; // Octal (គោលប្រាំបី) = 83
    $hexadecimal = 0x1A; // Hexadecimal (គោលដប់ប្រាំមួយ) = 26
    echo "Age: $age";
    echo "Binary 0b1010 = $binary";
    echo "Octal 0123 = $octal";
    echo "Hexadecimal 0x1A = $hexadecimal";
    echo "Max Integer: " . PHP_INT_MAX . "";
    echo "Min Integer: " . PHP_INT_MIN;
  • ?>
  • លទ្ទិផល.
    • Age: 25
    Binary 0b1010 = 10
    Octal 0123 = 83
    Hexadecimal 0x1A = 26
    Max Integer: 9223372036854775807
    Min Integer: -9223372036854775808

+ Compound Types.
(ប្រភេទទិន្នន័យស្មុគស្មាញ)

Compound Types គឺជាប្រភេទទិន្នន័យដែលអាចផ្ទុកតម្លៃច្រើនក្នុងពេលតែមួយ។
